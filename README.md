# Employee Management System API

A RESTful Web API built using:

- ASP.NET Core 8
- Entity Framework Core
- SQL Server
- Repository Pattern
- DTO Pattern
- AutoMapper
- Swagger
- Git & GitHub

## Features

- Create Employee
- Get All Employees
- Get Employee by ID
- Update Employee
- Delete Employee

## Technologies

- C#
- ASP.NET Core
- EF Core
- SQL Server
